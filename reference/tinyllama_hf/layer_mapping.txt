TinyLlama layer mapping
=======================

Narrative path:

```text
HF Transformer Reference
-> Minimal JAX Reproduction
-> StableHLO
-> IREE
-> Hardware Insight
```

The files below connect Hugging Face TinyLlama modules to small JAX
workloads and generated StableHLO artifacts.

```text

  LlamaMLP

  down_proj(silu(gate_proj(x)) * up_proj(x))

Minimal JAX:
  workloads/tinyllama/mlp_jax.py

StableHLO:
  ir/tinyllama/mlp/mlp.stablehlo.mlir

Internal primitive:
  workloads/tinyllama/swiglu_jax.py
  ir/tinyllama/swiglu/swiglu.stablehlo.mlir

Key StableHLO ops:
  stablehlo.dot_general
  stablehlo.multiply
  stablehlo.exponential
  stablehlo.divide

Hardware view:
  GEMM-heavy
  elementwise-heavy
  projection matmuls are systolic-array relevant
  SwiGLU gating is bandwidth/fusion sensitive

Status:
  implemented
```
