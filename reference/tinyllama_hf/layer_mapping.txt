TinyLlama layer mapping
=======================

Narrative path:

```text
HF Transformer Reference
-> Minimal JAX Reproduction
-> StableHLO
-> IREE
-> Hardware Insight
```

The files below connect Hugging Face TinyLlama modules to small JAX
workloads and generated StableHLO artifacts.

--------------------

Prefill attention score matmul

```text

  query_states @ key_states.transpose(-2, -1)
  scale by sqrt(head_dim)

  workloads/tinyllama/attention_scores_prefill_jax.py

StableHLO:
  ir/tinyllama/attention_scores_prefill/attention_scores_prefill.stablehlo.mlir

Key StableHLO ops:
  stablehlo.dot_general
  stablehlo.broadcast_in_dim
  stablehlo.divide

Hardware view:
  batched activation x activation GEMM
  score storage grows quadratically with sequence length
  causal mask and softmax are intentionally separate workloads

Status:
  implemented prefill score-only workload
```

Decode attention compute

```text
HF attention/cache step:
  compute one-token attention using updated visible K/V cache

Minimal JAX:
  workloads/tinyllama/attention_decode_jax.py

StableHLO:
  ir/tinyllama/attention_decode/attention_decode.stablehlo.mlir

Key StableHLO ops:
  stablehlo.dot_general
  stablehlo.reduce
  stablehlo.exponential
  stablehlo.divide

Hardware view:
  vector-matrix attention
  K/V cache read bandwidth matters
  M = 1 can underutilize large systolic arrays
  input K/V already include the current token position

Status:
  implemented decode compute-only workload
```

Decode K/V cache update

```text
HF attention/cache step:
  write new token K/V at cache_position during decode

Minimal JAX:
  workloads/tinyllama/attention_decode_cache_update_jax.py

StableHLO:
  ir/tinyllama/attention_decode_cache_update/attention_decode_cache_update.stablehlo.mlir

Key StableHLO ops:
  stablehlo.dynamic_update_slice
  stablehlo.compare
  stablehlo.select

Hardware view:
  small indexed cache write
  cache_position address generation is explicit
  output aliasing determines whether this becomes in-place

Status:
  implemented decode cache-update-only workload
```
