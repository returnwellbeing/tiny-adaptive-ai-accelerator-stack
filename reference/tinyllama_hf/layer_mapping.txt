TinyLlama layer mapping
=======================

Narrative path:

```text
HF Transformer Reference
-> Minimal JAX Reproduction
-> StableHLO
-> IREE
-> Hardware Insight
```

The files below connect Hugging Face TinyLlama modules to small JAX
workloads and generated StableHLO artifacts.

Implemented mappings
--------------------

Prefill attention score matmul

```text
HF attention step:
  query_states @ key_states.transpose(-2, -1)
  scale by sqrt(head_dim)

Minimal JAX:
  workloads/tinyllama/attention_scores_prefill_jax.py

StableHLO:
  ir/tinyllama/attention_scores_prefill/attention_scores_prefill.stablehlo.mlir

Key StableHLO ops:
  stablehlo.dot_general
  stablehlo.broadcast_in_dim
  stablehlo.divide

Hardware view:
  batched activation x activation GEMM
  score storage grows quadratically with sequence length
  causal mask and softmax are intentionally separate workloads

Status:
  implemented prefill score-only workload
```

```text

  prefill attention after q/k/v projection, reshape, transpose, and RoPE

Minimal JAX:
  workloads/tinyllama/attention_prefill_jax.py

StableHLO:
  ir/tinyllama/attention_prefill/attention_prefill.stablehlo.mlir

Key StableHLO ops:
  stablehlo.dot_general
  stablehlo.broadcast_in_dim
  stablehlo.reshape
  stablehlo.compare
  stablehlo.select
  stablehlo.reduce
  stablehlo.exponential
  stablehlo.divide

Hardware view:
  matrix-matrix prefill attention
  S x S score tensor growth
  softmax reduction over key sequence
  grouped-query K/V expansion or logical sharing
  no decode cache update in this workload

Status:
  implemented full prefill attention workload
```
